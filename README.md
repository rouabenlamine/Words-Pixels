# Words & Pixels

A Python-based search engine application with a PyQt5 interface.  
This project allows users to search for images and documents and displays results with similarity scoring.

## Features
- Search for images and documents using keywords
- Displays results in a simple PyQt5 GUI
- Calculates similarity scores for each result
- Works with local and API-based data sources

## Requirements
- Python 3.x
- PyQt5
- Requests (for API access, if used)
- Other dependencies: Pandas, etc.

Install dependencies using pip:

```bash
pip install pyqt5 pandas requests
